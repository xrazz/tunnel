# Contributing to Tunnel

Thanks for your interest! Here's how to contribute.

## Getting Started

1. Fork the repo
2. Clone your fork:
   ```bash
   git clone https://github.com/xrazz/tunnel.git
   cd tunnel
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start dev server:
   ```bash
   npm run dev
   ```

## Making Changes

1. Create a branch:
   ```bash
   git checkout -b feature/your-feature
   ```

2. Make your changes

3. Test it works:
   - Desktop to desktop
   - Desktop to mobile
   - Different file sizes

4. Commit:
   ```bash
   git add .
   git commit -m "feat: your feature description"
   ```

5. Push and open a PR:
   ```bash
   git push origin feature/your-feature
   ```

## Commit Messages

Use conventional commits:
- `feat:` new feature
- `fix:` bug fix
- `docs:` documentation
- `style:` formatting
- `refactor:` code refactoring

## What to Contribute

### High Priority
- Multiple file selection
- Resume interrupted transfers
- File compression
- QR code sharing
- Transfer history

### Nice to Have
- Dark mode
- Password-protected tunnels
- Custom tunnel codes
- Mobile app

## Bug Reports

Include:
1. What happened
2. What you expected
3. Steps to reproduce
4. Browser/OS info
5. Console errors (F12)
